# Speech to Text Project
# 🎙️ Speech-to-Text using Hugging Face

A Speech-to-Text application built using Hugging Face Transformers that converts audio files into text transcripts automatically.

## 🚀 Features

- Upload audio files for transcription
- Automatic speech recognition (ASR)
- Powered by Hugging Face pre-trained models
- Supports multiple audio formats
- Easy-to-use notebook implementation
- Fast and accurate transcription

## 🛠️ Tech Stack

- Python
- Hugging Face Transformers
- PyTorch
- Google Colab
- Jupyter Notebook

## 📂 Project Structure

```
speech-to-text/
│
├── huggingfacespeechtotext.ipynb
├── README.md
└── sample_audio.wav
```

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/Jayshree88patra/speech-to-text-.git
cd speech-to-text-
```

Install dependencies:

```bash
pip install transformers
pip install torch
pip install librosa
pip install soundfile
```

## ▶️ Usage

1. Open the notebook in Google Colab.
2. Install the required libraries.
3. Upload an audio file.
4. Run all cells.
5. View the generated transcript.

## 🧠 Model Used

This project uses Hugging Face Automatic Speech Recognition models such as:

- OpenAI Whisper
- Wav2Vec2

## 📊 Example

Input Audio:

```
hello.wav
```

Output:

```
Hello, welcome to speech recognition using Hugging Face.
```
